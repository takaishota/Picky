== Description ==

This plugin allows you to easily insert <a href="http://maps.google.com">Google
Maps</a> into your blog, making use of the new shortCode system in WordPress 2.5.
The maps can be configured to offer directions to or from the location, show or
hide the zoom/pan controls, show/hide map type (street view, satellite, etc),
activate zoom using mouse wheel, and more.  PHP5 Required.

== Installation ==

1. Verify that you have PHP5, which is required for this plugin.
1. Upload the whole `google-maps-for-wordpress` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Where do I insert a map into my post/page? =

When writing a post or page, there will be a meta box titled "Google Maps for
WordPress" just below the "Tags" and "Categories" metaboxes.

= Where can I get a Google Maps API Key? =

<a href="http://code.google.com/apis/maps/signup.html">http://code.google.com/apis/maps/signup.html</a>

= What unit do you expect the map width and height to be in? =

If you don't specify a unit, pixels will be used.  However, if you would like,
you can specify any valid CSS unit (em, %, etc).
